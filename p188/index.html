Running a 2nd instance of OpenVpn on DD-WRT 11/21/10 build 15778

On DDWRT Administration->Commands save the following script as Startup

sleep 40
mkdir /tmp/openvpncl2
sleep 5

# create config file

cat >/tmp/openvpncl2/vpn.conf <<EOL
ca /tmp/openvpncl2/ca.crt
cert /tmp/openvpncl2/client.crt
key /tmp/openvpncl2/client.key
verb 4
mute 5
log-append /var/log/openvpncl2
client
tls-client
dev tun1
proto udp
cipher bf-cbc
auth sha1
resolv-retry infinite
nobind
persist-key
persist-tun
mtu-disc yes
remote SERVERIP 1193
ns-cert-type server
fast-io
comp-lzo
EOL

# create ca.crt
sleep 5
cat >/tmp/openvpncl2/ca.crt <<EOL
-----BEGIN CERTIFICATE-----
-----END CERTIFICATE-----
EOL

# create client cert
sleep 5
cat >/tmp/openvpncl2/client.crt <<EOL
-----BEGIN CERTIFICATE-----
-----END CERTIFICATE-----
EOL

# create client key
sleep 5
cat >/tmp/openvpncl2/client.key <<EOL
-----BEGIN PRIVATE KEY-----
-----END PRIVATE KEY-----
EOL

openvpn --mktun --dev tun1
sleep 10
openvpn --daemon --config /tmp/openvpncl2/vpn.conf

Save the following commands as Firewall

  
iptables -A FORWARD -i tun1 -j ACCEPT
iptables -A FORWARD -o tun1 -j ACCEPT
iptables -I FORWARD -i br0 -o tun1 -j ACCEPT
iptables -I FORWARD -i tun1 -o br0 -j ACCEPT
