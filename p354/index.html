Pfsense 2.1beta OpenVPN notes.

Site 1: VPN server configuration

General information

Server Mode: Peer to Peer(SSL/TLS)

Protocol: UDP

Devide Mode: tun

Interface: WAN

Local Port: 1194

Cryptographic Settings

TLS Authentication: Enable authentication of TLS Packet.(paste shared key here. openvpn –genkey –secret ta.key) link.

Peer Certificate Authority: CA entry(ca.crt) from Pfsense Cert Manager. (./build-ca)

Peer Certificate Revocation list: CRL entry(crl.pem) from Pfsense Cert Manager.

Server Certificate: Server cert entry(VpnSrv.crt & VpnSrv.key) from Pfsense Cert Manager. (./build-key-server VpnSrv)

DH Parameters Length: 2048 bits

Encryption Algorithm: AES-256-CBC (256-bit)

Hardware Crypto: No Hardware Crypto Accelaration

Tunnel Settings

IPv4 Tunnel Network: 172.16.0.0/16

IPv4 Local Network: 10.0.0.0/8

IPv4 Remote Network: 10.0.0.0/8

Compression: Compress tunnel packets using the LZO algorithm

Advance configuration

ifconfig-pool-linear

For each vpn client create an entry Client Specific Override.

General information

Common Name: VpnClient1

Client Settings

Advanced: iroute 10.0.2.0 255.255.255.255 link.

Site 2 onwards: Client configuration

General information

Server Mode: Peer to Peer(SSL/TLS)

Protocol: UDP

Devide Mode: tun

Interface: WAN

Server host or address: or FQDN

Server Port: 1194

Server host name resolution: Enable Infinitely resolve name.

Cryptographic Settings

TLS Authentication: Enable authentication of TLS Packet.(paste shared key here. openvpn –genkey –secret ta.key) link.

Peer Certificate Authority: CA entry(ca.crt) from Pfsense Cert Manager. (./build-ca)

Client Certificate: Client cert entry(VpnClient1.crt & VpnClient1.key) from Pfsense Cert Manager. (./build-key VpnClient)

Encryption Algorithm: AES-256-CBC (256-bit)

Hardware Crypto: No Hardware Crypto Accelaration

Advanced configuration

remote-cert-tls server link

All Site : Firewall Rules

Interface: OpenVPN

TCPIP Version: IPv4

Protocol: any

Source: any

Destination: any
